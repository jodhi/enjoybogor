EnjoyBogor, Single Page Application for traveler in bogor built with Ionic framework.


front-end made with ionic framework v3
back-end made with php&mysql
wired with JSON  :) 

## Using this project

clone

```bash
$ git clone https://github.com/jodhi/enjoybogor
```

Then run:

```bash
$ cd enjoybogor
$ npm install
```
